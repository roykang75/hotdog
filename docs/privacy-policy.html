Hotdog Translator 개인정보처리방침

최종 수정일: 2026-03-04

1. 수집하는 정보

Hotdog Translator는 개인정보를 수집하거나 저장하지 않습니다.

  • 사용자의 브라우징 기록, 개인 데이터, 쿠키 등을 수집하지 않습니다.
  • 번역 대상 텍스트는 번역 API로 전송되지만, 확장 프로그램 자체에 저장되지 않습니다.
  • 사용자 설정(번역 엔진, 대상 언어, AI 서버 정보)은 chrome.storage.local에 로컬로만 저장됩니다.

2. 외부 서비스 통신

번역 기능을 위해 다음 외부 서비스와 통신합니다:

  • Google Translate API (translate.googleapis.com): 기본 번역 엔진으로 사용됩니다. 번역 대상 텍스트만 전송됩니다.
  • 사용자 지정 AI 서버: 사용자가 직접 등록한 OpenAI 호환 API 엔드포인트로 번역 요청을 보냅니다. 확장 프로그램은 해당 서버의 개인정보처리방침에 관여하지 않습니다.

3. YouTube 자막 기능

YouTube 시청 페이지에서 자막 번역 기능을 제공하기 위해, 확장 프로그램은 YouTube 페이지 내 네트워크 요청 중 자막 데이터(timedtext) 요청만을 감지합니다.

  • 이 기능은 오직 YouTube 도메인(youtube.com)에서만 동작합니다.
  • 자막 텍스트 이외의 네트워크 요청이나 사용자 데이터에는 접근하지 않습니다.
  • 감지된 자막 데이터는 번역 후 화면에 표시되며, 별도로 저장되거나 외부로 전송되지 않습니다.

4. 권한 사용 목적

  • activeTab: 사용자가 팝업에서 "번역" 버튼을 클릭한 현재 탭에만 접근합니다.
  • scripting: 번역 스크립트와 스타일을 현재 탭에 동적으로 주입합니다.
  • storage: 사용자 설정을 로컬에 저장합니다.
  • translate.googleapis.com: Google 번역 API 호출에 사용됩니다.

5. 데이터 보안

모든 번역 요청은 HTTPS를 통해 암호화되어 전송됩니다. 확장 프로그램은 서버를 운영하지 않으며, 사용자 데이터를 중앙에서 수집하거나 처리하지 않습니다.

6. 변경 사항

본 개인정보처리방침은 필요에 따라 업데이트될 수 있습니다. 변경 시 이 페이지에 최종 수정일과 함께 반영됩니다.

7. 문의

개인정보처리방침에 관한 문의는 GitHub 저장소의 Issues를 통해 접수할 수 있습니다.
